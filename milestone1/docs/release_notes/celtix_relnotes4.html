Installing Celtix

You can install Celtix on any Operating System that supports the Java 5 platform. To install Celtix Milestone 1, perform the following steps:

  1. Unpack the ZIP file into a directory of your choosing.

    Note:
    If you are unpacking the source distribution of Celtix, you must unpack it into a second-level directory (or deeper). For example, on Windows you could unpack it into C:\Programs\Celtix, but not C:\Celtix. This restriction does not apply to the binary distribution.

  2. Set the environment variable, CELTIX_HOME, to point to this directory.
