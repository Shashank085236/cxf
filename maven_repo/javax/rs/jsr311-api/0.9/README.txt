As jsr311-api.jar is not published in any public repository yet, you will need to install jsr311-api.jar 
by yourself in order to get JAX-RS implementation working. How to get this jar:

1. Download Jersey distribution from following link: https://jersey.dev.java.net/
2. Install Jersey, extract jsr311-api.jar from <Jersey-install-dir>/lib directory.
3. Rename jsr311-api.jar to jsr311-api-0.9.jar, put this jar under trunk\maven_repo\javax\rs\jsr311-api\0.9 directory.
